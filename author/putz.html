Groove Masta Putz - The Shit Show... for Mother Base

sáb 27 julio 2019

#music #demo

img source

Came home late last night and got inspired to write some raps. Decided I wanted to rap over The Base from Sonic Adventure 2 so I spruced it up and wrote about the Earth as a base. That's also why the title is how it is, cause all the songs on the soundtrack are named and for a specific level.

Hoping to use this momentum to start an album. Stay tuned.

EDIT: If you want the instrumental, click here. Had a buddy interested in rapping over it so I rendered one out.
